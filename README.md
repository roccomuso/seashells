> Seashells lets you pipe output from command-line programs to the web in real-time

Unofficial Node.js client for [Seashells.io](https://seashells.io).

## Install

    $ npm install -g seashells

## Usage

    $ echo "ciao" | seashells

    $ htop | seashells --delay 5
